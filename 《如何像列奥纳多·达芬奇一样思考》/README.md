# 保持好奇心
对周围的一切存在都要保持思考，思考它们是怎么来的，为什么会这样
伟大的思想会诞生出伟大的问题
我们提出的问题来自于我们的生活
它反映出我们的生活目标并且会影响我们的生活质量

## 好奇心应用和联系
+ 记日记或笔记本（最近记日记有点随意咧）
  + 记录问题，观察，想法的过程非常重要
  + 随身带在身上，也许现在可以用手机来代替，发送信息给notion
  + 镜像书写（->摩斯密码书写），可以更好地发现自己的问题
  + 不要担心顺序，只管记录
+ 在笔记本上列出100个重要的问题
  + 一口气完成
  + 不必担心重复，它会提醒重要的主题
  + 完成后通读，找出重点
